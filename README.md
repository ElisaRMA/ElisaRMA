## About me

Hi, I'm Elisa Ribeiro,
I'm a Senior Machine Learning Engineer at [Stone](https://www.stone.com.br/), with a PhD in Biology (computational biology focus) from [Unicamp](https://www.unicamp.br/unicamp/). 

I came into contact with R and programming at the end of my Master's, with the goal of improving the data analysis done in my research group. I ended up falling in love with programming and discovered the Data career path.

During my PhD, I associated [Metabolomics](https://www.ebi.ac.uk/training/online/courses/metabolomics-introduction/what-is/), plant biology, and Machine Learning for the quality control of Brazilian medicinal plants — which is also where I started exploring MLOps more seriously. I transitioned into the data field through Data Analyst and Data Scientist roles, and I'm now working as a Senior ML Engineer, building and maintaining production ML pipelines, including real-time anti-fraud systems and credit rating models.
